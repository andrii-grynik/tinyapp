# Dependencies
* bcrypt
* body-parser
* cookie - session
* EJS
* Express
* morgan
* Bode.js

#Test/Dev Dependencies
* chai
* mocha
* nodemon (as an option)

# Work Flow
* Insatll dependencies by using "npm istall" for modules. 
* Run web server with "node express_server.js" OR "npm start"(local directory)
